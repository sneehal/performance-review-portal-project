# Performance Review Portal Frontend

Beautiful responsive React frontend for your FastAPI + Flask AI backend.

## Run
```bash
cd frontend
npm install
npm run dev
```

Open: http://localhost:3000

## Backend URLs
Default:
- FastAPI: http://localhost:8000
- Flask AI: http://localhost:5000

Optional `.env`:
```env
VITE_API_BASE_URL=http://localhost:8000
VITE_AI_BASE_URL=http://localhost:5000
```

## Replace your empty `frontend` folder
Copy all files from this folder into:
`performance-review-portal/frontend`
